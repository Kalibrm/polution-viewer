
# Wymagania systemowe i zależności

## System operacyjny

- Linux (np. Ubuntu 22.04)
- Windows 10/11
- macOS 12+

## Wersje oprogramowania

- **Qt**: 6.8.2
- **CMake**: ≥ 3.16
- **C++**: Kompatybilny z C++17

## Zależności Qt

- QtBase
- QtQuick
- QtQuickControls2
- QtDeclarative
- QtGraphs

## Instalacja Qt (Linux)

Możesz zainstalować Qt za pomocą menedżera pakietów lub Qt Installer:

```bash
sudo apt install qt6-base-dev qt6-declarative-dev qt6-tools-dev cmake
```

### Instalacja Qt (Windows)

Pobierz i zaintaluj Qt 8.6.2 za pomocą [Qt Installer](https://www.qt.io/download)
